1. Find the details about "List" attribute in html5 forms. (<datalist>)
   The <datalist> tag specifies a list of pre-defined options for an <input> element. Users will see a drop-down 
   list of pre-defined options as they input data.The <datalist> element's id attribute must be equal to the 
   <input> element's list attribute always.
